# Rendertron Cache Server
Cache server which works as an intermediate between rendertron and the web.

## Usage
### 1. Clone repo
```
git clone git@bitbucket.org:experius/rendertron-cache-server.git
```

### 2. Setup environment
```bash
pip install -r requirements.txt
```

### 3. Run server
```bash
./start.sh
```

### 4. Open 
0.0.0.0:5000
